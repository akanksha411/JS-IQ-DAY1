3. What are the differences between var, let and const?

Ans-> var is introduced before ES6 Whereas let and const introduced on ES6 version of JavaScript.
      The scope of var is functional scope whereas the scope of let and const is block level
      var can be updated and redeclared in the scope. let can be uplated but not redeclarable. const can't be updated and redeclared.
      var and let can be declared without initialization but const can't

4. What is execution context?

Ans-> When browser encounters script tag, JS engine creates a new environment for execution. This environment is known as execution context.
           Two kinds of execution context
                i)GEC (Global Execution Context)
               ii)FEC  (Functional Execution Context)
            The creation of execuition context happens in two phase.
               i)Creation phase-->Memory allocation, variable etc happens here.
               ii)Execution Phase--> Code execution happens here.

5. What is meant by first class functions

Ans-> A programming language is said to have first class function when functions in that languages are treated like any other variable.
          JS has first class function,that can be
               i) Assigned to variables

                        function printHi(){

6. What are closures? Give an example of closure

Ans-> Closure is a function that has acess to the variables in its lexical environment even when it is executed outside the environment.
            function makeCounter(){
                let count=0;
                return function(){
                    count +=1;
                    return count;
                }
            }
            const counter=makeCounter();
            console.log(counter());
            console.log(counter());

Explain call(), apply() and, bind() methods. Give an example of call(), apply(), bind()

Ans-->The call method is used to call a function with a specified this value and arguments passed as individual values. 
            Example:  
            function test(arg1,arg2){
                console.log(this.name,arg1,arg2);
            }
            const testName={
                name:'Prepbytes'
            }
            test.apply(testName,100,200)
      The apply method is similar to call() but arguments passed as array.  
            Example:
            function test(...arguments){
                console.log(this.name,arguments);
            }
            const testName={
                name:'Prepbytes'
            }
            test.call(testName,[100,200])
       The bind method is used to create a new function, that has the same body of the original function, but with a specified this value and arguments. 
            Example
            function test(arg){
                console.log(this.name,arg);
            }
            const testName=test.bind(name:'Prepbytes','EA17')
            testName()
